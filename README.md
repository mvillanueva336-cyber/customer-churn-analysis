# Customer Churn Analysis

Author: Max Villanueva  
MBA in Business Analytics (In Progress)

## Project Overview

This project analyzes telecom customer data to identify factors that contribute to customer churn. Customer churn occurs when a customer stops using a company's services, and understanding churn is critical for improving customer retention and long term business performance.

The analysis uses real telecom customer data and applies exploratory data analysis techniques to identify patterns and relationships between customer characteristics and churn behavior.

## Objectives

The goal of this project is to answer several key business questions:

• What percentage of customers churn  
• Which contract types have the highest churn rates  
• Whether monthly charges influence customer churn  
• Whether customer tenure affects the likelihood of churn  
• Which payment methods correlate with higher churn rates

## Dataset

The dataset used in this project contains telecom customer information including demographic characteristics, account details, services subscribed, and whether the customer churned.

Key variables include:

• tenure  
• contract type  
• internet service  
• monthly charges  
• payment method  
• churn status

• Data loading and inspection

• Customer tenure versus churn analysis  
• Payment method churn rate evaluation  
• Data visualization using Matplotlib and Seaborn

## Key Insights

Initial analysis suggests that:

• Month to month contracts show the highest churn rates  
• Customers with higher monthly charges tend to churn more frequently  
• Customers with shorter tenure are more likely to leave  
• Payment method and billing preferences may influence churn behavior

These insights help illustrate how companies can identify at risk customers and improve retention strategies.
